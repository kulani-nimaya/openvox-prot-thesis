# Speech Rate Feedback Tool for Dysarthria Therapy

## Overview
Desktop application for measuring speech rate and providing feedback...

## Requirements
- Java 8+
- Praat executable
- Microphone

## Installation
1. Clone repository
2. Place Praat in `/tools/` folder
3. Run `java -jar SpeechRateTool.jar`

## Usage
1. Click "Start Recording"
2. Speak for 5 seconds
3. View feedback on speech rate
